Lost Tech TensorFlow Cloud Edition Privacy Policy


Lost Tech TensorFlow Cloud Edition collects the following information:

- information about physical or virtual hardware the Software is running on
- amount of time spent using the Software
- cost of renting the hardware from the cloud

This information is used to calculate the correct bill amount for the usage
of the Software and may be processed and stored by your cloud provider. We
never share this information with third parties.


You can also explicitly opt-in to share the following information with Lost Tech LLC:

- types of objects you create and use with TensorFlow

This information is used to improve future versions of Lost Tech TensorFlow.
It is never shared with third parties.
